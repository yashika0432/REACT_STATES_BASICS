## ⚛️ React State Basics
A beginner-friendly project demonstrating the fundamentals of React state using simple examples.
This repository helps understand how state works, updates, and triggers re-renders in React components.

---

## 📌 About the Project
State is one of the core concepts in React. It allows components to store and manage dynamic data that changes over time.

This project demonstrates:
- What state is in React
- How to create state using the useState hook
- How state updates trigger UI re-rendering
- Managing and updating state in functional components
The examples are intentionally simple to help beginners understand the core concept easily.

---

## 🚀 Features
- Beginner-friendly React examples
- Demonstrates useState hook
- Interactive UI to show state updates
- Clean and minimal code structure
- Easy to extend for learning purposes
